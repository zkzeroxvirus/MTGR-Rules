## 1. In Game Resources

:package: There are two resources used in MTG Roguelike: Essence and XP.

:sparkles: Essence

Essence is your long-term progression currency.
It persists between sessions and is spent before or after runs to unlock permanent power.

There are two separate progression profiles:

@Verified Host
Trusted hosts running the Base table. Progression here is shared across all Verified Hosts.
Your Crypt Buffs, Tickets, Brands, Captures, Achievements, and stored Essence carry over between verified tables.

@Host
Hosts using the Unverified Essence Counter for custom modes and alternate table styles.
Progress from these sessions is separate and does not mix with Verified progression.

How Essence is gained:

- Crypt win reward: +500 Essence per eligible player
- Crypt win reward: +Deck CMC + Sideboard CMC per eligible player
- Unspent XP at run end converts into Essence, win or lose
- If a Crypt boss Buff awarded on win is already unlocked, that duplicate is immediately sold for +250 Essence

:large_orange_diamond: XP

XP is your in-run currency.
It is earned from encounters and spent during the run, mainly in Town and Wanderer systems.

How XP works:

- Base XP comes from encounter stage and mode
- Affixes add bonus XP
- Some encounters add extra bonus XP
- XP does not persist as XP between sessions
- Unspent XP converts to Essence at run end

Rule of thumb:
Spend XP to power up your current run.
Spend Essence to power up future runs.

## 3. Building Your Deck

:one: Roll Your Commander

- Generate 5 random legal Commanders
- Choose 1
- Each player has 4 Commander Mulligans before Encounter 1
- Extra Commander rerolls only come from specific effects

:two: Generate Your Pool and Draft 22

- Generate your own 100-card pool inside your Commander color identity
- Pool can contain any card type
- Basic lands are excluded
- Stickers, Attractions, and Contraptions are rerolled and replaced
- Draft 22 cards from that pool

Reroll rules:

- You cannot reroll the entire 100-card pool
- If one card is invalid, replace only that card
- Commander rerolls are still allowed (lock applies only to deck-card draft pool)

Color identity note:

- Initial deckbuilding is color-identity locked
- During the run you can still get off-color cards through systems like Cashouts and Mystery Packs

:three: Add Lands

- Deck uses 16 total lands
- +2 duals at 2 colors
- +4 duals at 3 colors
- +6 duals at 4 colors
- +8 duals at 5 colors
- Remaining lands are basics

:four: Add Your Free Choice Card
Choose 1 card with these restrictions:

- Must follow Commander color identity
- Not banned in Commander
- Not Sol Ring
- Not a Gamechanger

After selecting:

- Stamp with a Scryfall decal

Combo restrictions:

- No two-card combos in any run deck (players or Host)
- Scryfall-stamped cards cannot be part of a three-card combo

:five: Final Checks

- Final deck is Commander + 39
- Sideboard is allowed
- All pre-game effects must be locked before start
- Apply progression loadout (Buffs, Achievements, Tickets, Captures) within slot limits
- Resolve pregame Event then Trinket (starting Event cannot be a Bad Stuff Event)
- After deckbuilding and before Encounter 1 begins, players may make one seat-swap adjustment; after Encounter 1 begins, seat swapping is no longer allowed

## 4. Basics of the Game

Once deckbuilding is done, every run follows the same loop.

Core loop:

- Encounter
- Rewards
- Town or Stay Out choice
- Event resolution
- Next Encounter

Run modes:

- Standard: 3 encounters, then Crypt
- Variant: 4 encounters, then Crypt

Party structure:

- Players function as one coordinated party
- Host controls encounters and opposition

Win and fail:

- You win by defeating the Crypt
- Run fails only if the party fails to defeat the Crypt
- Unspent XP converts to Essence either way

Death rules:

- Death is not permanent
- No base XP penalty on death
- Optional death penalties are Host discretion only

## 5. Encounters and Rewards

Each encounter follows:

1) Determine encounter type
2) Resolve setup
3) Apply Affixes
4) Set Host HP
5) Play encounter

Encounter type notes:

- Encounter 1 rerolls Tribal Warfare and Story Fight
- Story Fight and Wild Magic include special bonus/restriction rules
- Crypt uses no Affixes, Host always goes first, and no Events resolve there

Rewards on victory:

- XP (mode base + Affix bonus + encounter-specific bonus)
- Cashout
- Loot Pool

On defeat:

- No XP
- No Cashout
- No Loot Pool
- Party still proceeds to Town or Stay Out

Cashout resolution:

- Host reveals player count + 1 Cashouts
- Each player picks one (unless otherwise stated)
- Unchosen options are removed
- Cashouts are free and tradable

Loot Pool resolution:

- Reveal cards equal to player count from defeated deck
- Reroll basic lands
- Revealed cards form a shared pool players may take from freely

Affix bonus XP by tier:

- Tier 1: +5
- Tier 2: +10
- Tier 3: +15
- Tier 4: +20

Cashout Pool (weight 410) categories:

- Pack Rewards (300)
- Bonus Building Usage (80)
- Blacksmith Services (18)
- Crypt Fight Cashout (2)
- Ticket Cashouts (10)

## 6. Turn Based Rules

📘 How Turns Work (Party vs. Host)

🛡️ Party Turn
The party acts as one shared player with one combined turn:

- Untap
- Upkeep
- Draw
- Main 1
- Combat
- Main 2
- End Step

Upkeep and end-step triggers occur once per party turn, not once per player.
Controller-specific effects still benefit only their controller.

⚔️ Party Combat

- All attackers are declared together
- Host blocks after full declaration

👹 Host Turn

- Host plays a normal Magic turn

⚔️ Host Combat

- Host chooses attack targets
- Adjacent players can block for attacked adjacent allies
- A player cannot use adjacent-blocking to block their own creature when they attacked an adjacent player

⏩ Extra Turns

- Only the controller of the effect takes the extra turn
- Crypt rule always overrides first-player effects: Host goes first in Crypt

## 8. Town and Town Actions

🏘️ Town is the controlled progression path between encounters.

When entering Town:

- Fully heal all players
- Resolve Town building usage
- Resolve Event(s) and continue loop

Town does not use actions.
Buildings are either limited-use or unlimited.

Town Buildings:

🏦 Bank (1 use per Town)

- Move up to 6 cards between deck and sideboard

⚒️ Blacksmith (Unlimited)

- 25 XP Upgrade
- 50 XP Augment
- One Blacksmith modification per card

🛍️ Bazaar (2 Deals per Town + unlimited Cashout Sales)
Deals:

- Card <-> Card
- Card <-> Essence (2x mana value)
- Cashout <-> Cashout
- Cashout <-> Card
Cashout Sales:
- Sell any number of Cashouts for 10 XP each

⛪ Cathedral (1 use per Town)

- Describe a request (cannot force one exact card)
- Host provides real usable card
- Add to deck or destroy for Essence equal to 2x mana value

⚔️ The Guild (Unlimited)

- 10 XP + d6
- Receive roll+2 random Commanders in declared color combination
- May swap Commander
- Old Commander goes to deck/sideboard
- Deck must remain legal under new color identity

🛒 Merchant (Unlimited)

- Open 15-card pack, keep 1
- 5 XP Mystery
- 10 XP ID
- 20 XP Pro
- 50 XP Mythic
- 75 XP OTAG

🔮 Mystic (Unlimited)

- 5 XP
- Destroy nonland card from deck
- Gain Essence equal to 2x mana value

🌀 Portal (1 use per Town)

- Spend up to 15 XP
- Generate pack by type/keyword criteria, size = XP spent
- Generated cards are Temporal

🍺 Tavern (1 use per Town)
Option A:

- Remove 0-2 cards
- Add 0-4 basics
Option B:
- Remove 3-5 basics
- Add 0-2 duals

## 9. Trinkets and How to get them

🔮 Trinkets are pre-Encounter 1 run modifiers.

Timing and selection:

- Resolve Event, then Trinket before Encounter 1
- Starting Event cannot be Bad Stuff

## 10. Crypt and End Game

💀 The Crypt is the final encounter of every run.

Locked Crypt rules:

- No Affixes
- Host takes first turn
- No Events are resolved

Run end:

- Win: run completes. Each eligible player gains 500 Essence, plus Deck CMC + Sideboard CMC, plus the awarded Crypt boss Buff (duplicate Buffs are auto-sold for +250 Essence). Unspent XP converts to Essence.
- Lose: run fails, unspent XP still converts to Essence

Persistent progression is tracked across runs and includes:

- Crypt Buffs
- Tickets
- Brands
- Achievements
- Captures

Sample Crypt Buff unlocks:

- The God Trees Blessing
- Shapeshifter
- Treasure Pirate

Sample achievement tracks:

- General Achievements
- Creature-Type Achievements
- Guild Identity Achievements

For full unlock requirements and effect text, use the Permanent Progression document.
