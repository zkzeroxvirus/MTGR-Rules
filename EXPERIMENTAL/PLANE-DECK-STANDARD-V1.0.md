# **🧪 Experimental Plane Deck Standard V1.0**

This experimental standard defines how Plane, Set, or Block-based decks are built.

## **Purpose**

This format is intended for the Deckbuilding team when the Host pilots a Plane, Set, or Block-based commander deck.

## **Deckbuilding Rules**

• Standard deck size is **100 cards including the commander**
• The commander must come from the chosen Plane / Set / Block
• Cards that originate from that same Plane / Set / Block may appear up to **4 copies each**
• Cards that originate outside that Plane / Set / Block are limited to **1 copy each**
• At least **75%** of the full deck list must originate from the chosen Plane / Set / Block
• For the standard **100-card deck**, this means at least **75 cards** must originate from the chosen Plane / Set / Block

## **Notes**

• This is an experimental rule set and is not part of the core deckbuilding rules
• If this standard is adopted later, it can be promoted into `CORE-RULES.md`
